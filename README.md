# vaskeliste

## Setup

```bash
# install dependencies
$ npm install

# serve with hot reload at localhost:3000
$ npm run dev
```

## Build

```bash
# deploy to gh-pages
$ npm run deploy

# build public site
$ npm run generate

# build dynamic site
$ npm run build

```

For detailed explanation on how things work, check out the [documentation](https://nuxtjs.org).
